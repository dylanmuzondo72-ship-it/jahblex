# Jah Blex Editorial Profile Website

A responsive, static editorial website for Jah Blex (Tawananyasha Katsande), using the supplied photographs and publicly indexed social/profile information.

## Files

- `index.html` — complete one-page website
- `assets/styles.css` — responsive design and layout
- `assets/script.js` — mobile navigation and gallery lightbox
- `assets/images/` — supplied photographs

## Run locally

Open `index.html` in a browser, or serve the folder with any static server.

## Deploy

### Vercel
1. Create a new GitHub repository and upload all files in this folder.
2. Import the repository into Vercel.
3. Keep the framework preset as **Other** and deploy.

### Netlify
Drag the entire folder into Netlify Drop, or connect the GitHub repository.
